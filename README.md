## 高仿bilibiliAPP

### 技术栈
前端  vue + axios + webpack + scss

后端  Express

### 目前实现的功能包括：
1. 首页推荐

2. 分类滑动导航

3. 搜索

4. 视频播放 （双击播放/暂停， 进度条拖动，倍速播放，全屏播放）

5. 相关视频推荐

6. 评论

7. 弹幕

8. 一键换肤

弹幕效果现在全部内容是从右向左滚动的效果，等后续再实现高级效果

项目启动

```
后端启动
bilibili-vue/bilibili-express 目录下
npm start

前端启动
bilibili-vue 目录下
npm install 
npm run dev
```
先启动后台接口，再启动前端
